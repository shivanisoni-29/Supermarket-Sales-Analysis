# Supermarket-Sales-Analysis

## Project Overview

This project analyzes supermarket sales data using Python and performs Exploratory Data Analysis (EDA) to discover business insights.

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Project Workflow

- Data Loading
- Data Cleaning
- Exploratory Data Analysis
- Data Visualization
- Correlation Analysis
- Business Insights
- Conclusion

## Key Insights

- Highest revenue branch identified
- Best-selling products analyzed
- Product category performance evaluated
